# 🐼 PandaHSK - Ứng dụng học tiếng Trung HSK

**Lớp:** CT1B  
**Nhóm:** Group 12

## 📝 Giới thiệu dự án
PandaHSK là ứng dụng hỗ trợ người dùng học và ôn tập từ vựng tiếng Trung theo cấp độ HSK (1-6). Ứng dụng kết hợp các phương pháp học hiện đại giúp người dùng ghi nhớ từ vựng hiệu quả thông qua Flashcards và hệ thống tra cứu đa nguồn.

## ✨ Tính năng chính
- **Tra cứu từ điển đa năng:** Kết hợp API từ Thivien, Tatoeba và MyMemory để cung cấp âm Hán Việt, nghĩa tiếng Việt và ví dụ song ngữ.
- **Quản lý Flashcard cá nhân:** Cho phép người dùng tự tạo bộ thẻ từ riêng, sửa và xóa linh hoạt.
- **Học theo cấp độ:** Phân loại bài học theo trình độ HSK từ 1 đến 6.
- **Nhập/Xuất dữ liệu (CSV):** Hỗ trợ sao lưu và nạp dữ liệu từ file Excel/CSV một cách nhanh chóng.
- **Chế độ tối (Dark Mode):** Giao diện hiện đại, bảo vệ mắt.
- **Hoạt động Offline:** Hỗ trợ lưu trữ đệm (Cache) giúp xem lại bài học khi không có mạng.

## 🛠 Công nghệ sử dụng
- **Ngôn ngữ:** Kotlin
- **UI Framework:** Jetpack Compose (Material Design 3)
- **Kiến trúc:** MVVM + Clean Architecture
- **Backend:** Firebase (Firestore, Authentication)
- **Thư viện:** Retrofit (API), Coroutines (Xử lý bất đồng bộ), SharedPreferences (Quản lý Session).

## 🚀 Hướng dẫn cài đặt
1. Clone dự án về máy: `git clone https://github.com/UHL-FIT/ltttbdd-btl-20261-ct1b_group12.git`
2. Mở dự án bằng **Android Studio (Ladybug hoặc mới hơn)**.
3. Đợi Gradle đồng bộ (Sync).
4. Nhấn **Run** để chạy trên máy ảo hoặc thiết bị thật (Android 8.0 trở lên).
